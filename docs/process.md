# Montage Campaign Process

## Overview

This document outlines the workflow and key steps involved in running a campaign in Montage.

## Roles

- Jurors
- Coordinators
- Organizers

## Process

- Campaign creation  
  - Define start date and end date → determines entry eligibility range  

- First round creation and import  
  - Define start date and end date → determines juror voting window  

- Round types  
  - Rating  
  - Yes/No  
  - Ranking  

- Advancing to the next round  
- Final ranking round  
- Closing a campaign  
- Publishing the report  
- Exporting results
